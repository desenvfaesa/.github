# .github

Repositório especial da organização **FAESA** no GitHub.

Ele centraliza arquivos e configurações compartilhados entre todos os repositórios da organização, como:

- [`profile/README.md`](profile/README.md) — README institucional exibido na página de perfil da organização
- Templates de issues e pull requests
- Arquivos de configuração e automações (workflows) reutilizáveis
- Diretrizes de contribuição aplicáveis a todos os projetos da FAESA

Para conhecer mais sobre a FAESA, acesse o [perfil da organização](profile/README.md) ou o [site oficial](https://www.faesa.br).
